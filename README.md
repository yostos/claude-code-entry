非エンジニア向けの Claude Code セルフハンズオン資料です。

架空の経営支援サービス会社の社員として、Web サイトリニューアルの提案依頼書（RFP）を Claude Code と一緒に作成します。文書作成の 5 つの工程（構想 → 仕様 → 執筆 → 校正 → リリース）を通じて、エージェント型 AI との協業を体験できます。

## 対象者

- プログラミング経験がない、または少ない方
- 生成 AI を業務の文書作成に活用したい方

## 前提条件

- Windows PC
- Markdown の基本的な書き方を学習済み（参考: [Markdown 記法一覧](https://gist.github.com/mignonstyle/083c9e1651d7734f84c99b8cf49d57fa)）
- Claude Pro/Max プランの契約、または API クレジット

## ダウンロード

[Releases](https://github.com/yostos/claude-code-entry/releases) ページから最新版の zip ファイルをダウンロードし、展開してください。

## 使い方

1. ダウンロードしたフォルダ内の `handson.html` をブラウザで開く
2. スライドに沿って進める

## ファイル構成

```
handson.md              # ハンズオンスライド（Marp 形式）
handson.html            # ビルド済みスライド（ブラウザで閲覧）
resources/
  textlintrc.json       # 校正ツール（textlint）の設定ファイル
```

## 所要時間

約 3 時間 40 分（自分のペースで進められます）
